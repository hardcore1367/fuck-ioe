# Flipflop

A **flip-flop** is a basic memory element in sequential circuits that stores one bit of data (0 or 1) and changes its output based on the clock signal and input.

### Truth Table

| EN | S | R | $Q_{n+1}$ | Remarks |
| --- | --- | --- | --- | --- |
| 0 | x | x | Qn | - |
| 1 | 0 | 0 | Qn | Hold |
| 1 | 0 | 1 | 0 | Reset |
| 1 | 1 | 0 | 1 | Set |
| 1 | 1 | 1 | ? | Invalid |

### Characteristic Table
| S | R | $Q_n$ | $Q_{n+1}$ | Remarks |
| --- | --- | --- | --- | --- |
| 0 | 0 | 0 | 0 | Hold |
| 0 | 0 | 1 | 1 | Hold |
| 0 | 1 | 0 | 0 | Reset |
| 0 | 1 | 1 | 0 | Reset |
| 1 | 0 | 0 | 1 | Set |
| 1 | 0 | 1 | 1 | Set |
| 1 | 1 | 0 | x | Invalid |
| 1 | 1 | 1 | x | Invalid |

### Excitation Table
| $Q_n$ | $Q_{n+1}$ | S | R |
| ----- | --------- | - | - |
| 0     | 0         | 0 | X |
| 0     | 1         | 1 | 0 |
| 1     | 0         | 0 | 1 |
| 1     | 1         | X | 0 |
