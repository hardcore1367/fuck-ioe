## 2.2 Signal Analysis and Processing

**Signal Analysis and Processing** means studying and modifying signals to extract useful information or improve their quality.

---

1. **Time Reversal (Folding / Inversion)**
    - Signal is **flipped around the y-axis**.
    - Equation: $y(t) = x(-t)$
    
    <img width="742" height="284" alt="image" src="https://github.com/user-attachments/assets/cb04e8ed-42ef-4b1d-9d25-fd0360c9cb9b" />

    
2. **Time Shifting**
    - Signal is **moved left or right** in time.
    - Equation: $y(t) = x(t - t_0)$
    - **Right shift:** t₀ > 0 → delayed signal
    - **Left shift:** t₀ < 0 → advanced signal

    <img width="609" height="395" alt="image" src="https://github.com/user-attachments/assets/a4a68563-294f-40c4-8db6-441f51e3d0fe" />

    
3. **Time Scaling**
    - **Compresses or expands** the signal in time.
    - Equation: $y(t) = x(a t)$
    - If a > 1 → signal **compressed (faster)**
    - If a < 1 → signal **expanded (slower)**
    
    <img width="598" height="369" alt="image" src="https://github.com/user-attachments/assets/427e2792-8901-4357-bb37-f6c4f2b1d044" />

    
4. **Amplitude Transformation**
    - Amplitude transformation means **changing the height or level of a signal** by multiplying it with a constant and/or adding another constant.
    - Equation: $y(t) = A  x(t) + B$
    
    **Conditions:**
    
    1. **A > 1** → Signal amplitude **increases** (amplified)
    2. **0 < A < 1** → Signal amplitude **decreases** (attenuated)
    3. **A = -1** → Signal **inverted** (flipped vertically)
    4. **A = 0** → Output becomes a **constant B**
    5. **B > 0** → Signal shifts **upward**
    6. **B < 0** → Signal shifts **downward**
    
    **In short:**
    
    - **A** controls **amplitude scaling and inversion**.
    - **B** controls **vertical shifting** of the signal.
5. **Combination of Reversal and Shifting**
    - Apply reversal first, then shift:
        
        $y(t) = x(-t + t_0) = x[t_0 - t]$
        
    - Used to mirror and move signals together.
6. **Combination of Time Shifting and Scaling**
    - Apply scaling first, then shift:
        
        $y(t) = x(a t + b)$

## 2.2 Classification of Signals

### 1. Continuous and Discrete-Time Signal

### 1. Continuous-Time Signal

1. **Definition:** Signal defined for every instant of time.
2. **Representation: $x(t),$**, where *t* is continuous.
3. **Graph:** Smooth and continuous curve.
4. **Property:** Value exists at all *t*.

<img width="640" height="231" alt="image" src="https://github.com/user-attachments/assets/3a0b211a-88e2-4bfd-871c-253c65bf867a" />


### 2. Discrete-Time Signal

1. **Definition:** Signal defined only at discrete time intervals.
2. **Graph:** Sequence of separate points.
3. **Property:** Obtained by sampling a continuous-time signal.

<img width="666" height="275" alt="image" src="https://github.com/user-attachments/assets/381f3d2e-d556-489c-9342-84f9e4f98066" />


### 2. Periodic and Non-Periodic Signal

### Periodic Signal

1. **Definition:** Repeats itself after a fixed time period T.
2. **Property:** Continuous repetition over time.

---

### Non-Periodic (Aperiodic) Signal

1. **Definition:** Does not repeat over any period of time.
2. **Property:** Occurs only once or changes continuously without repetition.

### 3. Deterministic and Random Signal

### Deterministic Signal

1. **Definition:** Can predict exact value at any time.
2. **Condition:** Value known by a mathematical expression.
3. **Property:** Fully predictable and reproducible.

---

### Random Signal

1. **Definition:** Cannot predict exact value at any time; only statistical properties known.
2. **Condition:** Value changes unpredictably.
3. **Example:** Noise signal, stock prices.
4. **Property:** Characterized by **mean, variance, correlation**.

### 4. Energy and Power Signal

### **Difference (Energy vs Power)**

| Feature | Energy Signal | Power Signal |
| --- | --- | --- |
| Total Energy | Finite | Infinite |
| Average Power | 0 | Finite |
| Examples | Exponential decay, pulse | Sine wave, periodic signals |
| Occurrence | Aperiodic | Periodic or random |

### 5. Causal, Anti-Causal and Non-Causal

### Causal System

1. **Definition:** Output depends only on **present and past inputs**.
2. **Property:** Can be physically realized; no future input needed.

---

### Anti-Causal System

1. **Definition:** Output depends only on **future inputs**.
2. **Property:** Cannot be physically realized; relies on future values.

---

### Non-Causal System

1. **Definition:** Output depends on **past, present, and future inputs**.
2. **Property:** Cannot be implemented in real-time systems.

### 6. Even and Odd Signals

### Even Signal

1. **Definition:** Symmetric around vertical axis; ( x(t) = x(-t) ).
2. **Property:** Mirror image on left and right sides of y-axis.

---

### Odd Signal

1. **Definition:** Anti-symmetric around vertical axis; ( x(t) = -x(-t) ).
2. **Property:** Left side is inverted mirror of right side.

### 7. Analog and Digital Signal

### Analog Signal

1. **Definition:** Continuous in **time and amplitude**.
2. **Property:** Can take **any value** at any time.
3. **Example:** Voice, sine wave, radio signal.

---

### Digital Signal

1. **Definition:** Discrete in **time or amplitude** (or both).
2. **Property:** Represented by **0s and 1s**; less affected by noise.
3. **Example:** Computer data, digital audio, binary signals.

---

## 2.3 System

### Continuous-Time System

1. **Definition:** Input and output signals are **continuous in time**.
2. **Property:** Works with continuous-time signals like analog signals.

---

### Discrete-Time System

1. **Definition:** Input and output signals are **defined only at discrete time instants**.
2. **Property:** Works with sampled or digital signals.

### Linear System

1. **Definition:** Obeys **superposition principle** (additivity + homogeneity).
2. **Equation:**
    - Additivity: $f(x_1 + x_2) = f(x_1) + f(x_2)$
    - Homogeneity:  $f(a x) = a f(x)$
3. **Property:** Output is directly proportional to input.

---

### Nonlinear System

1. **Definition:** Does **not obey superposition**; output is not proportional to input.
2. **Property:** Can produce distortion, harmonics, or complex behavior.

### Time-Invariant System

1. **Definition:** Output does **not change** if input is shifted in time.
2. **Property:** Behavior is the same at all times.

---

### Time-Variant System

1. **Definition:** ~~Output **changes** if input is shifted in time.~~
2. **Property:** System behavior depends on the time of input.

### Static (Memoryless) System

1. **Definition:** Output depends **only on present input**.
2. **Property:** No memory of past or future inputs.

---

### Dynamic (With Memory) System

1. **Definition:** Output depends on **past and/or future inputs**.
2. **Property:** Has memory; stores information about input history.

### Causal System

1. **Definition:** Output depends only on **present and past inputs**.
2. **Property:** Can be physically realized; real-time implementation possible.

---

### Non-Causal System

1. **Definition:** Output depends on **future inputs** (or past, present, and future).
2. **Property:** Cannot be implemented in real-time; needs future input values.

### Invertible System

1. **Definition:** Input can be **recovered exactly** from output using an inverse system.
2. **Property:** Has a one-to-one mapping between input and output.

---

### Non-Invertible System

1. **Definition:** Input **cannot be recovered** exactly from output.
2. **Property:** Mapping is not one-to-one; information is lost.

### Stable System

1. **Definition:** **Bounded input produces bounded output (BIBO stable)**.
2. **Property:** Output never goes to infinity for a finite input.

---

### Unstable System

1. **Definition:** **Bounded input can produce unbounded output**.
2. **Property:** Output may grow infinitely for a finite input.
