## 1.1 Data and Signal

- **Data:** An entity that conveys meaning based on mutually agreed rules between sender and receiver.
- **Signal:** The physical form of data used to transmit it over a medium — can be electrical, optical, or electromagnetic.

## 1.4 Communication System

### 1. Analog Communication System

An analog communication system transmits information using continuous signals that vary in amplitude or frequency. The input (such as voice) is converted to an electrical signal by a transducer, then modulated by the transmitter to make it suitable for channel transmission. At the receiving end, the signal is demodulated to recover the original message and converted back to its physical form.

**Main Elements:**

1. **Input Transducer:** Converts physical quantity → electrical signal (e.g., microphone).
2. **Transmitter:** Processes message into transmittable form; key operation is **modulation**.
3. **Channel:** Physical medium (wire, air, fiber) linking sender and receiver.
4. **Receiver:** Recovers message from received signal; key operation is **demodulation**.
5. **Output Transducer:** Converts electrical signal → physical output (e.g., speaker).

**Flow:**
Source → Input Transducer → Transmitter → Channel → Receiver → Output Transducer → Destination

### 2. Digital Communication System

A digital communication system transmits information as discrete binary data (0s and 1s). The source data is first encoded to remove redundancy, then channel-coded to add error protection, and finally modulated for channel transmission. At the receiver, these operations are reversed — demodulation, error correction, and decoding — to faithfully recover the original data. Digital systems are preferred over analog because they offer better noise immunity, error correction, and security.

**Main Elements:**

1. **Source:** Generates digital data (text, image, audio, etc.)
2. **Source Encoder:** Converts data to binary; removes redundancy (compression).
3. **Channel Encoder:** Adds error detection/correction bits.
4. **Modulator:** Converts bits into a signal suitable for the channel.
5. **Channel:** Transmission medium; signal may be corrupted by noise here.
6. **Demodulator:** Converts received signal back to binary.
7. **Channel Decoder:** Detects and corrects errors using added bits.
8. **Source Decoder:** Reconstructs original data from binary.
9. **Destination:** Final receiver of the information.

**Flow:**
Source → Source Encoder → Channel Encoder → Modulator → Channel → Demodulator → Channel Decoder → Source Decoder → Destination


### 3. Analog vs Digital System

| **Feature** | **Analog** | **Digital** |
|---|---|---|
| Signal Type | Continuous (sine wave) | Discrete (0s and 1s) |
| Noise Effect | Highly affected | Less affected |
| Transmission Quality | Degrades over distance | Maintained via regenerators |
| Error Detection | Difficult | Easy (error codes) |
| Security | Less secure | Highly secure (encryption) |
| Example | Radio, TV | Internet, mobile |

## 1.5 Transmission Impairments

Transmission impairments are factors that degrade signal quality during transmission over a channel. The three main types are attenuation, distortion, and noise.

**1. Attenuation:**
Loss of signal strength as it travels through the medium over distance. The signal becomes weaker and may become unreadable. Fixed using **amplifiers** (analog) or **repeaters** (digital).

**2. Distortion:**
The shape or form of the signal changes during transmission. Types:
- **Amplitude distortion** — different frequencies amplified unequally.
- **Delay distortion** — different frequencies arrive at different times.
- **Phase distortion** — phase shift introduced in the signal.

Fixed using **equalizers**.

**3. Noise:**
Unwanted signals that mix with the original signal, corrupting it. Types:
- **Thermal noise** — caused by random electron movement due to heat; always present.
- **Intermodulation noise** — signals on different frequencies mix and produce unwanted signals.
- **Crosstalk** — signal from one channel leaks into another.
- **Impulse noise** — sudden, short-duration spikes caused by external events (e.g., lightning).

## 1.6 Performance Parameters of Communication Channel

**1. Bandwidth (BW)**
- Range of frequencies a channel can carry.
- Unit: Hz
- Higher BW → higher data rate possible.
- Example: Voice channel BW = 4 kHz

**2. Throughput**
- Actual amount of data successfully transmitted per unit time.
- Unit: bps
- Always ≤ channel capacity (due to overhead, errors, congestion).
- Throughput < Bandwidth always in practice.

**3. Latency (Delay)**
- Total time for first bit to travel from sender to receiver.
- Components:
  - **Propagation delay** = Distance / Propagation speed
  - **Transmission delay** = Message size / Bandwidth
  - **Queuing delay** = time waiting in buffer
  - **Processing delay** = time to process at node

**4. Jitter**
- Variation in packet arrival time.
- Caused by variable queuing/propagation delays.
- Critical for real-time applications (VoIP, video streaming).
- High jitter → poor audio/video quality.

**5. Bit Error Rate (BER)**
- Ratio of incorrectly received bits to total transmitted bits.
- Formula: `BER = Number of error bits / Total bits transmitted`
- Lower BER → better channel quality.
- Affected by noise, attenuation, interference.


## 1.7 Thermal Noise Power Density

**Thermal noise** (Johnson noise) is caused by random motion of electrons due to heat.

**Formula:**
$$N_0 = k \cdot T \quad \text{(W/Hz)}$$

Where:
- $k$ = Boltzmann's constant = $1.38 \times 10^{-23}$ J/K
- $T$ = Temperature in Kelvin = °C + 273

**Total noise power over bandwidth B:**
$$N = k \cdot T \cdot B \quad \text{(Watts)}$$

**Solved Example (from 2071 set):**

> Calculate thermal noise power density at T = 17°C

$$T = 17 + 273 = 290 \text{ K}$$
$$N_0 = 1.38 \times 10^{-23} \times 290$$
$$N_0 = 4.002 \times 10^{-21} \text{ W/Hz}$$

$$N_0 \approx 4 \times 10^{-21} \text{ W/Hz}$$

**Key points to remember:**
- Thermal noise is **white noise** — flat spectrum across all frequencies.
- It sets the **fundamental lower limit** on noise in any system.
- Cannot be eliminated — only reduced by lowering temperature.
- At room temperature (17°C), $N_0 \approx 4 \times 10^{-21}$ W/Hz.
