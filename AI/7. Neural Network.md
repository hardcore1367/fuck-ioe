# Neural Network

An artificial neural network (ANN) is a computational model inspired by the biological neural networks in the human brain, consisting of interconnected nodes called artificial neurons. These networks are used in machine learning to recognize patterns and make predictions based on input data.

### Components of Neural Network Structure

**1. Inputs (xᵢ)**
- Raw data values fed into the network
- Each input represents one feature (e.g., pixel value, temperature)

**2. Weights (wᵢ)**
- Each connection has a weight — represents its importance
- Higher weight = stronger influence on output
- These are what the network **learns/adjusts** during training

**3. Bias (b)**
- Extra constant added to net input
- Shifts the activation threshold — lets neuron fire even when all inputs are 0
- Think of it like the y-intercept in `y = mx + c`

**4. Summation (Σ)**
- Computes: `net = (w₁x₁ + w₂x₂ + ... + wₙxₙ) + b`
- Combines all weighted inputs into a single value

**5. Activation Function**
- Takes `net` and decides the output
- Adds **non-linearity** — without it, the network is just linear math
- Examples: Step, Ramp, Sigmoid

**6. Output (y)**
- Final result of the neuron
- Passed to next layer or used as the network's answer
