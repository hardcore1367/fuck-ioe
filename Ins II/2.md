## Mode 1

| Port    | Mode 1 Input  | Mode 1 Output |
| ------- | ------------- | ------------- |
| **PC7** | —             | OBF̅ (Port A) |
| **PC6** | —             | ACK̅ (Port A) |
| **PC5** | IBF (Port A)  | —             |
| **PC4** | STB̅ (Port A) | —             |
| **PC3** | INTR (Port A) | INTR (Port A) |
| **PC2** | STB̅ (Port B) | ACK̅ (Port B) |
| **PC1** | IBF (Port B)  | OBF̅ (Port B) |
| **PC0** | INTR (Port B) | INTR (Port B) |


## MODe 2
| Port C bit | Signal   | Description                         |
| ---------- | -------- | ----------------------------------- |
| **PC7**    | **OBF̅** | Output Buffer Full (A → Peripheral) |
| **PC6**    | **ACK̅** | Acknowledge from peripheral         |
| **PC5**    | **IBF**  | Input Buffer Full (Peripheral → A)  |
| **PC4**    | **STB̅** | Strobe from peripheral              |
| **PC3**    | **INTR** | Interrupt request for Port A        |
| **PC2**    | —        | Free / usable for Port B            |
| **PC1**    | —        | Free / usable for Port B            |
| **PC0**    | —        | Free / usable for Port B            |
