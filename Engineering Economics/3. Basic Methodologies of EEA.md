MARR - Minimum Acceptable Rate of Return

### 1. Payback Period Method
#### 1. Simple Payback Period
> Does not consider Time Value of Money

Steps to Solve:
1. Create the table below
2. Cumulative CF = Current Net CF + Previous Year's Cumulative CF
3. Payback period lies where Cumulative CF turns positive (interpolate)
$SPP = Sign\ Changed\ Year + \frac{|Last\ negative\ Cumulative\ CF|}{Positive\ Amount\ from\ CF\ in\ current\ Year}$ 

| EOY | Net Cash Flow | Cumulative Cash Flow |
|-----|---------------|----------------------|
| 0   | Given         | = Net CF             |
| 1   | ...           | ...                  |
| n   | ...           | ...                  |

#### 2. Discounted Payback Period
> Same as SPP but Cumulative column uses Present Worth of each CF

| EOY | Net Cash Flow | Present Worth | Cumulative PW |
|-----|---------------|---------------|---------------|
| 0   | Given         | ...           | ...           |
| 1   | ...           | ...           | ...           |
| n   | ...           | ...           | ...           |

### 2. Equivalent Worth Methods
> Idea: Convert all CF to a single point using MARR, > If $EW > 0$ → accept

1. Present Worth Method

   $Equivalent\ Worth = Total\ CF\ In_{at\ PW} - Total\ CF\ Out_{at\ PW}$

2. Future Worth Method

   $Equivalent\ Worth = Total\ CF\ In_{at\ FW} - Total\ CF\ Out_{at\ FW}$

3. Annual Worth Method
   
   $AW = Revenue_{AW} - Expense_{AW} - CR$
   
   $CR = Investment_{AW} - Salvage_{AW}$

### 3. Rate of Return Methods
> If $> MARR$ → accept
#### 1. IRR - Hit and Trial
1. Choose an equivalent worth method (e.g. PW)
2. Set $PW = 0$, $i$ = unknown, and create an equation
   > HINT: Solve the equation using calculator to find the value, so that we can guess more clearly.
3. Guess some value of $i$ and put them in above equation
4. Guess $i_1$ → calculate PW → should be **positive**
5. Guess $i_2$ → calculate PW → should be **negative**
6. Interpolate:

$$IRR = i_1 + \frac{|PW_1|}{|PW_1| + |PW_2|} \times (i_2 - i_1)$$


#### 2. Unrecovered Investment Balance 

$$UIB_n = UIB_{n-1}(1+i) + CF_n$$

> $UIB_{n-1}(1+i)$ = FW of previous year's balance brought forward

| Year | CF | UIB |
|------|----|-----|
| 0 | ... | ... |
| n | ... | ... |

> IRR is the value of $i$ at which $UIB_n = 0$

#### 3. ERR
If investment rate is given, use it while converting to PW/FW, else use MARR
1. Move all `-ve` CF to PW (year 0)
2. Move all `+ve` CF to FW (year n)

$$ PW_{costs}(1+ERR)^n = FW_{benefits}$$

### 4. Benefit Cost Ratio
> If $BCR > 1$ → accept

$$BCR = \frac{FW(B)}{FW(I) - FW(S) + FW(O/M)}$$

$$BCR_M = \frac{FW(B) - FW(O/M)}{FW(I) - FW(S)}$$

- $B$ = Benefits
- $I$ = Initial Investment
- $S$ = Salvage Value
- $O/M$ = Operation & Maintenance Cost
> Same goes for PW, AW
