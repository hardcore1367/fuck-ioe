## Chapter 8 — Project Time Management

**Type:** Numerical + Theory (Numerical dominant — highest weightage chapter)

```
Type 1 — CPM/Network Diagram (deterministic durations)
Frequency: 13
Pattern:
  Given: Table of activities with predecessor(s) and fixed duration (days/weeks)
  Find: Network diagram, EST/EFT/LST/LFT, critical path, total/free/independent/interfering float, project completion time
Sub-types: AON (Activity-on-Node) tables; AOA (Activity-on-Arrow) with node numbers; forward/backward pass only
⚠️ Note: No two datasets are identical, but the ask (draw diagram → floats → critical path) repeats almost verbatim every set — this is the single most important numerical type to master.

Type 2 — PERT (probabilistic, 3-time estimates)
Frequency: 5
Pattern:
  Given: Optimistic, Most Likely, Pessimistic time for each activity
  Find: Expected time (te), variance/SD, expected project completion time, probability of completion within X weeks/years using Z-score & normal distribution
Sub-types: probability within a stated duration; probability of a specific activity finishing by a certain week
⚠️ Note: appears roughly once every 4 sets — recurring but not identical data.

Type 3 — Crashing
Frequency: 1 (2081 Bhadra — "Robotics Club boat racing" problem)
Pattern:
  Given: Activity time, max crash time, crash cost, crash slope
  Find: Regular network diagram + total duration; modified (crashed) network diagram + additional crash cost incurred
Sub-types: none seen repeated
⚠️ Note: Low frequency but high marks (12) when it appears — worth knowing the method even though it's asked rarely.
```

## Chapter 9 — Project Cost Management

**Type:** Numerical + Theory (Numerical dominant)

```
Type 1 — Earned Value Management (EVM)
Frequency: 14
Pattern:
  Given: Planned Value (PV/BCWS), Earned Value (EV/BCWP), Actual Cost (AC/ACWP) OR % complete + budget + amount spent
  Find: CV, SV, CPI, SPI, EAC, ETC, VAC; conclude ahead/behind schedule, over/under budget
Sub-types: reverse problems (SPI given → find PV); "earned value is twice actual cost" style; EVA-graph illustration
⚠️ Note: This is asked in nearly every single paper (14/22) — the single highest-frequency numerical type in the whole subject. Master every variant of the EVM formula set.

Type 2 — NPV / Payback Period (capital budgeting)
Frequency: 1
Pattern:
  Given: Initial project cost, cash flow per year, interest/discount rate
  Find: Payback period, NPV, accept/reject decision
Sub-types: none repeated
⚠️ Note: rare but distinct from EVM — don't confuse the two.
```
